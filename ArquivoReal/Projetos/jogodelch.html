Kamikaze Man







O processo desse projeto

Neste projeto os alunos tiveram total liberdade para criar um projeto em python com a única condição de que fosse ou um jogo ou um site. A liberdade de criação deu aos alunos um incentivo para querer aprender e até pesquisar temas antes mesmo de serem dados em aula. A grande maioria dos projetos surpreendeu a todos positivamente com jogos com um grau de complexidade relativamente alto.

KamikazeMan foi um projeto inspirado no clássico BomberMan, começamos do zero criando classes orientadas a objete e inclusive desenhando os sprites. Adquirimos um alto conhecimento de classes, funções e looping dado que tivemos a necessidade de criar um contador de frames para fazer o clock das bombas.
