The Legend od Ayres







O processo desse projeto

Neste projeto os alunos tiveram total liberdade para criar um projeto em python com a única condição de que fosse ou um jogo ou um site. A liberdade de criação deu aos alunos um incentivo para querer aprender e até pesquisar temas antes mesmo de serem dados em aula. A grande maioria dos projetos surpreendeu a todos positivamente com jogos com um grau de complexidade relativamente alto.

Meu projeto foi um jogo de rpg, realizado utilizando a biblioteca pygame do python, pela qual fomos capazes de criar um jogo com uma serie de funcionalidades, display gráfico, diversos oponentes e alguns diferentes senários. O código do jogo foi construído inteiro em classes onde cada classe realizava uma série de funções características como demonstrado na foto. O jogo foi chamado de “The Legendo of Ayres” em homenagem ao nosso querido professor de design de software Fabio Ayres. Tivemos de aprender por completo como utilizar a biblioteca pygame, função de Clock entre outras para conseguir cumprir o desafio de escrever o código de um jogo completamente do 0.
