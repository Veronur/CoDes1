Simulação de um Sistema Mecânico







O processo desse projeto

Neste projeto os alunos foram dados a tarefa de modelar um sistema mecânico, ainda sem conceitos rigorosos de derivada ou integral. Utilizaram somente equações a diferenças, diferenciais e a função ODEINT no python. Os projetos variaram desde simulações de pontos materiais até corpos rígidos. O projeto foi muito interessante pois ao final do mesmo os alunos perceberam que conseguiram modelar situações um grau de complexidade bem elevado, envolvendo arrasto, rotações e até três dimensões, sem definições formais de derivada e integral.

Para o meu projeto escolhemos modelar o sistema de bungee jump assim por meio de fórmulas físicas construímos nosso modelo e o simulamos por meio do Python. Como pergunta escolhemos responder para diferentes pesos qual deveria ser a altura mínima de instalação para um bungee jump.
